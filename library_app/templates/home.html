About our links above:

Home you can enter our homepage. (Thats this webpage!)

Add New Borrower you can enter details to add a new borrower to our library database.

Add New Book you can enter details to add a new book to our library database.

Book Search you can search for books currently in our library database. The search can be used with any book title, isbn, or author name.

Check In Book you can quickly check in your book using your Card ID and Book Loan ID.

Borrower Search you can search for borrowers in our library database. The search can be used with any borrower name, card ID, loan ID, or phone number.

Loan Search you can search for book loans in our library database. The search can be used wth any borrower name, card ID, loan ID, or book isbn.
